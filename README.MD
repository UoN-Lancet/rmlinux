# RMOSS Linux

### RMOSS Linux is a customized linux distro based on Zorin OS 16 (Ubuntu 20.04). It integrates development environments for c++, java, python, C#, rust and other languages as well as sdk for ROS, OpenCV, OpenVino, etc., intend to provide a more user-friendly development environment for beginners.
